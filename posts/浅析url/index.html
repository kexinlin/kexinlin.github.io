浅析URL

2020年8月3日

URL

URL 全程为 Uniform Resource Locator，包括的部分有：协议、域名或 IP、端口号、路径、查询字符串、锚点。

  • 协议：规定请求的格式、响应的格式
  • 域名或 IP：定位设备
  • 端口号：定位服务
  • 路径：请求不同页面
  • 查询字符串：请求同一个页面不同内容
  • 锚点：请求同一个内容的不同位置

DNS 的作用

DNS 全称为 Domain Name System。

在终端上键入nslookup <域名>，会展示通过 DNS 获取到的域名对应 IP。

IP 的作用是什么

IP 的作用是用于定位设备。

在终端上键入ping <域名>，会展示请求该域名时的对应 IP。

域名是什么，分别哪几类域名

域名是 IP 的别称。

域名分为顶级域名、一级域名、二级域名、三级域名，它们之间是父子关系。
