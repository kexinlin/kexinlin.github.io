a 标签的用法

a标签的主要作用是：

  1. 跳转外部页面
  2. 跳转内部锚点
  3. 跳转到邮箱或电话等

a标签的属性有href、target、download、rel=noopener。

跳转到外部网址：

href属性的值可以是 https://<网址>，或者 http://<网址>，或者//<网址>。

例子：

<a href="https://www.google.com">谷歌</a>

用//<网址>的好处是，在不确定是 https 还是 http 协议的情况下，这样不会错，因为会自动选到正确的。

跳转到路径：

href属性的值可以是绝对路径也可以是相对路径。

这里绝对路径的根目录就不是硬盘根目录了，而是哪里开的服务，哪里就是根目录。

例子：

<a href="./index.html">谷歌</a>

跳转内部锚点：

内部锚点要配合id一起使用。

例子：

<p id="xxx">This is a paragraph</p>
<a href="#xxx">查看xxx</a>

跳转到邮箱和电话：

把href属性的值写成 mailto:<邮箱地址>，就会自动调取发邮件功能并填写该地址。

把href属性的值写成 tel:<电话号码>，就会自动调取打电话功能并使用该号码。

例子：

<a href="mailto:kexin.lin@mail.utoronto.ca">发邮件给我</a>
<a href="tel:15012868179">打电话给我</a>

img 标签的用法

img标签

img标签的作用：发出 GET 请求，展示一张图片。

img标签的属性：src / alt / height / width。

其中，src 属性后面填写图片链接，alt 属性后面填写当 src 属性连接 GET 失败后展示的内容。

height / width 属性：如果只写一个属性，图片大小会自适应。如果写两个，图片可能变形。

例子：

<img src="dog.png" height="500px" , alt="./404.png" />

img标签的事件：onload / onerror。

onload 和 onerror 是 JavaScript 中的两个事件，用来监听图片是否加载成功。

响应式：max-width。 max-width 可以用来适应手机屏幕大小。

例子：

<style>
  img {
    max-width: 100%;
  }
</style>
<img src="dog.png" height="500px" , alt="./404.png" />

table 标签的用法

table标签相关的标签有：table、thead、tbody、tfoot、tr、td、th。

例子：

<table>
  <thead>
    <tr>
      <th></th>
      <th>小红</th>
      <th>小黄</th>
      <th>小兰</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>语文</td>
      <td>86</td>
      <td>72</td>
      <td>82</td>
    </tr>
    <tr>
      <td>数学</td>
      <td>94</td>
      <td>97</td>
      <td>89</td>
    </tr>
    <tr>
      <td>英语</td>
      <td>96</td>
      <td>91</td>
      <td>92</td>
    </tr>
  </tbody>
</table>

如果我们没有写thead、tbody、tfoot，直接写tr和td的话，html 会默认用tbody补全。

如果我们给表格加上边框，每个格子之间都能看到间隙，非常丑。解决方案是用border-collapse和border-spacing来调整。

例子：

<style>
  table {
    border-collapse: collapse;
    border-spacing: 0;
  }
  td,
  th {
    border: 1px solid blue;
  }
</style>
