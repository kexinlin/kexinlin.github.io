HTML 是谁发明的

HTML的发明者是Tim Berners-Lee，也叫李爵士。他写了第一个浏览器、第一个服务器，用自己写的浏览器访问了自己写的服务器。他发明了WWW，同时发明了URL、HTML以及HTTP。 2017年，李爵士获得图灵奖。

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<body>
    
</body>
</html>

章节标签

章节标签主要用来表示文章/书的层级，下面介绍几个比较常见的。

h1~h6：表示标题，其中数字越小表示层级越高。在HTML的默认样式中，标题会被加粗。

section：表示段落。

article：表示文章。

p：表示段落。

header：表示头部，和footer对应。

footer：表示尾部，和header对应。

main：表示主要内容，和aside对应。

aside：表示旁支内容，和main对应。

div：表示划分。

全局属性

全局属性是所有标签都有的属性，下面介绍几个常见的。

class：表示分类，需要注意的是在style中，.xxx可以用来匹配class名称中空格分隔的一部分，而[class=xxx]则是精准匹配。

contenteditable：表示元素可以被用户所编辑。

hidden：表示元素被隐藏，页面上不可见。

id：有说法是如果是全局唯一就用id，如果不是唯一用class，但这一说法并不算准确。建议是不到万不得已不要用id，原因一：id不唯一时不报错；原因二：如果id名覆盖了专用名词，用js控制该id的时候就会失效（比如，如果有id叫parent，parent.style.border = "10px solid green"就不管用）。

style：用于指定样式，优先级高于css（另外，js会覆盖style，因此优先级最高）。

tabindex：指定按tab键访问到的顺序，从小到大。另外注意，tabindex=0是tab访问的最后一个，tabindex=-1是tab永远访问不到的。

title：鼠标放上去后浮出展示的完整内容。

内容标签

ol + li：表示ordered list + list item。

ul + li：表示unordered list + list item。

dl + dt + dd：表示description list + term + data。

pre：表示preview。html的一个特点是，连续的多个空格或是回车或是tab会被视为一个空格，但用pre标签围绕的时候就不会。

code：一般用来围绕代码。用code标签的好处是字体是等宽字体（monospace），符合代码字体的规则。

hr：水平分割线。

br：换行。

a：超链接，注意target="_blank"指明链接在新窗口打开。

em：语气重要的强调，默认样式为斜体。

strong：本质重要的强调，默认样式为加粗。

quote：表示非换行引用。

blockquote：表示块级引用。
