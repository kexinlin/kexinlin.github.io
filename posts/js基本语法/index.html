什么是表达式和语句

  • 表达式：一般有值
  • 语句：可能有值也可能没有，一般会改变环境（声明、赋值）

表达式的例子：

1 + 2

语句的例子：

a = 3

标识符的规则

第一个字符可以是Unicode字母或$或中文，后面的字符除了前面的之外还可以有数字。

例子：

var_ = 1
_var = 1
$var = 1
var1 = 1

if else 语句

语法： if(表达式){语句1}else{语句2}

完整推荐写法：

if (表达式) {
    语句
} else if (表达式) {
    语句
} else {
    语句
}

while for 语句

语法： while(表达式){语句}

用法：表达式为真，执行语句，执行完再判断表达式的真假；表达式为假，不执行语句，执行再之后的语句。

break continue

  • break：退出所有循环
  • continue：推出当前一次循环

label

label：表示一个代码块（或代码行）的标签。

例子：

foo: 1

foo: {
    console.log(1);
    a = 1;
    break foo;
}
