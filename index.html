JS对象基本用法

2020年11月4日
声明对象的两种语法 第一种语法： let obj = {a: 123} 第二种语法： let obj = new Object({a: 123}) 如何删除对象的属性 第一种语法： let obj = {a: 123} delete obj.a 第二种语法： let obj = {a: 123} delete obj['a'] 如何查看对象的属性 // 查看所有属性 let obj = {a: 123} Obj.keys(obj) // 查看自身属性和共有属性 console.dir(obj) // 查看特定属性 obj['a'] obj.a 如何修改或增加对象的属性 // 直接增加或修改 let obj = {a: 123} obj['a'] = 456 obj['b'] = 123 obj.b = 456 ‘name’ in obj和obj.hasOwnProperty(‘name’) 的区别 'name' in obj返回true，说明'name'属性可能是obj的自身属性，也可能是共有属性。 obj.hasOwnProperty('name')返回true，说明'name'属性是obj的自身属性。……
